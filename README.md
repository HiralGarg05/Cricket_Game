 🏏 Cricket Game (Bat-Ball-Stump)

A simple and fun web-based cricket game built using HTML, CSS, and JavaScript.

 🎮 Game Description

This is a mini game inspired by the classic "Bat-Ball-Stump" concept:

* User selects one option: **Bat**, **Ball**, or **Stump**
* Computer randomly selects one option
* The winner is decided based on predefined rules

 🧠 Game Logic

* Bat beats Ball
* Ball beats Stump
* Stump beats Bat
* Same choice → It's a tie

 🚀 Features

* Interactive buttons for user input
* Random computer choice using JavaScript
* Instant result display using alerts
* Simple and beginner-friendly code

🛠️ Technologies Used

* HTML
* CSS
* JavaScript
